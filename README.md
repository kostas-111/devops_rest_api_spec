# Внешнее API

## Описание 

В проекте создана спецификация через Swagger OpenAPI в формате YAML.
Следуем принципу - документация как источник истины

## Применение

- Можем сгенерировать серверную часть. Это можно сделать через сайт https://editor.swagger.io/ или через клиента openapi-generator-cli.
- Импортировать спецификацию в Postman. Он позволит выполнить запросы к серверу.
- Можно сгенерировать заглушки на серверную сторону для вызова клиента.
- Возможность написать тесты по спецификации, которые будут проверять ваше API. Это позволит внести дополнительную проверку в согласование API.
